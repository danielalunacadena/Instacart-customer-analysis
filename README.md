# Instacart
Data analysis project using Python to clean, analyze, and visualize customer purchasing behavior and reorder patterns in an online grocery platform.

# Instacart Customer Behavior Analysis

## Project Overview
This project analyzes customer purchasing behavior using Instacart grocery delivery data.  
The objective is to clean the data and generate insights into shopping habits, reorder patterns, and product performance.

## Data
- Orders, products, aisles, and departments datasets
- Large datasets with missing and duplicate values

## Key Tasks
- Data preprocessing (handling missing and duplicate values)
- Exploratory data analysis
- Customer behavior analysis
- Product popularity and reorder rate analysis
- Data visualization

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Identified peak shopping times and days
- Analyzed reorder behavior across products
- Highlighted top-performing and frequently reordered products
